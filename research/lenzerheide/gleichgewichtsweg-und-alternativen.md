# Gleichgewichtsweg & Alternativen in der Lenzerheide

Recherche für den Familienbesuch im August 2026 (Nichte 6,5 Jahre, Grosseltern 70 Jahre).
Hintergrund: Der Churer Gleichgewichtsweg war letzten Sommer ein Highlight. Einen offiziellen Gleichgewichtsweg gibt es in der Lenzerheide nicht, aber drei Angebote kommen dem Erlebnis nahe. Der Originalweg in Chur liegt zudem direkt auf der An-/Abreiseroute.

---

## Das Original: Churer Gleichgewichtsweg (Fürstenwald, Chur)

| | |
|---|---|
| **Ort** | Fürstenwald, Chur (oberhalb der Stadt) |
| **Beschreibung** | Ca. 450 m langer Parcours mit 19 Balancier-Hindernissen im Wald, Motto «der Boden ist Lava», für alle Generationen |
| **Eignung Kind (6,5 J.)** | Perfekt, bereits erprobt und geliebt |
| **Eignung Grosseltern** | Gut, Hindernisse sind freiwillig, Waldweg daneben begehbar |
| **Preis** | Gratis |
| **Öffnungszeiten** | Jederzeit frei zugänglich |
| **Praktisch** | Liegt quasi auf der Anreiseroute Ellwangen–Lenzerheide (via Chur). Ideal als Zwischenstopp am Anreisetag oder am Abreisevormittag |
| **Link** | <https://die-buergergemeinde.ch/tipps/churer-gleichgewichtsweg/> |

---

## Alternative 1: Autschliweg (Scharmoin → Lenzerheide Dorf) ⭐ Beste Alternative

| | |
|---|---|
| **Ort** | Start Mittelstation Scharmoin (Rothornbahn), Ziel Dorf Lenzerheide |
| **Beschreibung** | Themenweg mit dem magischen Bären Autschli: Herausforderungen meistern, Rätsel lösen und auf Baumstämmen balancieren. Autschli versteckt sich unterwegs hinter Bäumen |
| **Eignung Kind (6,5 J.)** | Sehr gut, Balancieren und Rätsel treffen genau ihre Interessen |
| **Eignung Grosseltern** | Gut, der Weg führt bergab Richtung Dorf |
| **Preis** | Weg gratis, Bergbahn auf Scharmoin kostenpflichtig |
| **Bonus** | Direkt an der Mittelstation Scharmoin: die grösste Murmelbahn (Kugelbahn) der Schweiz mit über 250 Laufmetern, plus LIKEaBIKE-Park und Erlebnisland |
| **Link** | <https://arosalenzerheide.swiss/de/Lenzerheide/Sommer/Familienerlebnisse/Wandern-mit-Kindern> |

---

## Alternative 2: Spielplatz Heidsee (Lenzerheide)

| | |
|---|---|
| **Ort** | Am Heidsee, Lenzerheide/Valbella |
| **Beschreibung** | Grosser Spielplatz mit Piratenschiff, Hängebrücke, Kletterfelsen und Spielbach, viele Balancier- und Kletterelemente |
| **Eignung Kind (6,5 J.)** | Sehr gut |
| **Eignung Grosseltern** | Sehr gut, Kiosk mit Snacks und warmen Gerichten sowie Grillstelle direkt nebenan, Sitzgelegenheiten am See |
| **Preis** | Gratis, Baden im Heidsee ebenfalls gratis |
| **Praktisch** | Kinderwagentauglich, gut kombinierbar mit Pedalo/Baden am See |
| **Link** | <https://arosalenzerheide.swiss/de/Lenzerheide/Sommer/Familienerlebnisse/Spielplaetze> |

## Alternative 3: Wetterwichtelweg (Heidbüel, Churwalden)

| | |
|---|---|
| **Ort** | Heidbüel oberhalb Churwalden |
| **Beschreibung** | Rätselweg mit 10 Stationen: mit einem Blickrichter Tiere entdecken, die Buchstaben verraten. Die entschlüsselte Wetterbotschaft kann an der Kasse im Portal Churwalden oder im Infobüro Lenzerheide gegen eine kleine Überraschung eingetauscht werden |
| **Eignung Kind (6,5 J.)** | Sehr gut, Rätsel plus Belohnung am Ende |
| **Eignung Grosseltern** | Gut, gemächliches Tempo durch die Stationen |
| **Preis** | Weg gratis, Bergbahn Heidbüel kostenpflichtig |
| **Kombi-Tipp** | Am selben Hang wie der Erlebnisberg Pradaschier: Sommerrodelbahn (längste der Schweiz) direkt anschliessbar |
| **Link** | <https://arosalenzerheide.swiss/de/Lenzerheide/Sommer/Familienerlebnisse/Wandern-mit-Kindern/Wetterwichtelweg-Heidbueel_tour_19523175> |

## Empfehlung für die Planung

- **Tag 1 oder Tag 3:** Churer Gleichgewichtsweg als Zwischenstopp auf der Fahrt (bekannt und beliebt, gratis, flexibel)
- **Tag 2:** Autschliweg als Lenzerheide-Pendant, kombiniert mit Murmelbahn und Erlebnisland Scharmoin
- **Flexibel/spontan:** Spielplatz Heidsee als hotelnaher Dauerbrenner
- **Kombi-Option:** Wetterwichtelweg + Sommerrodelbahn Pradaschier als Halbtagesprogramm in Churwalden
